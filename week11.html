Week 11: JS Conditionals & Loops

This week focuses on **decision making and loops** in JavaScript, letting your page respond dynamically to conditions and repeat tasks efficiently.

Day 1: If Statements

Learn: Use if to execute code only when conditions are true. See:

let age = 16;
if (age >= 18) {
  console.log("You can vote");
} else {
  console.log("Too young to vote");
}
Do: Create a variable for your age and log a message based on if you’re a teenager or not. Check: Console displays correct message.

Day 2: Else If & Nested Conditions

Learn: else if handles multiple conditions, nested if allows complex logic. See:

let score = 85;
if (score >= 90) {console.log("A");}
else if (score >= 80) {console.log("B");}
else {console.log("C");}
Do: Create grade messages for three ranges. Check: Console prints correct grade.

Day 3: For Loops

Learn: Loops repeat code. See:

for (let i = 1; i <= 5; i++) {
  console.log("Number " + i);
}
Do: Log numbers 1–10 using a for loop. Check: Verify numbers in console.

Day 4: While Loops

Learn: while loops run while a condition is true. See:

let i = 1;
while (i <= 5) {
  console.log(i);
  i++;
}
Do: Use a while loop to log even numbers between 2 and 20. Check: Console shows all even numbers.

Day 5: Break & Continue

Learn: break exits a loop early, continue skips to the next iteration. See:

for (let i=1; i<=10; i++) {
  if (i===5) break;
  console.log(i);
}
for (let i=1; i<=5; i++) {
  if (i===3) continue;
  console.log(i);
}
Do: Create a loop that skips multiples of 3 and stops at 12. Check: Verify numbers printed in console.

Day 6: Combining Conditionals & Loops

Learn: Combine loops and conditionals to create dynamic checks. See:

for (let i=1; i<=10; i++) {
  if (i % 2 === 0) console.log(i + " is even");
  else console.log(i + " is odd");
}
Do: Log numbers 1–20 and state whether each is even or odd. Check: Console outputs correct results.

Day 7: Review & Mini Interactive Script

Learn: Combine loops and conditionals to solve small problems. Do: Create a program that asks for a number and prints all numbers from 1 to that number with "Fizz" if divisible by 3, "Buzz" if divisible by 5, or "FizzBuzz" if divisible by both. Check: Browser console shows correct outputs. This completes Week 11.

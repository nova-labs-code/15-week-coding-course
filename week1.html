Week 1: HTML Basics

Welcome to Week 1! This week focuses on the foundations of HTML. HTML structures everything you see on a webpage. Each day follows a simple flow: learn, see, practice, and check your work.

Day 1: HTML Overview & Page Structure

Learn: HTML is the backbone of web pages. Every HTML file starts with <!DOCTYPE html> and is wrapped in <html>. Metadata goes inside <head> and visible content goes inside <body>.

See:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first paragraph.</p>
</body>
</html>

Do: Create a new HTML file. Add your name inside an <h1> tag and write a paragraph about yourself.

Check: Open it in a browser and refresh after editing to see changes instantly.

Tip: Indentation makes your code readable but does not affect how browsers display it.
Day 2: Headings

Learn: Headings organize content from <h1> to <h6>. Only one <h1> per page.

See:

<h1>My Website</h1>
<h2>About Me</h2>
<h3>Hobbies</h3>

Do: Create a page with your name as H1, "About Me" as H2, and a "Hobbies" subsection as H3.

Check: Headings should decrease in size logically.

Day 3: Paragraphs & Text Formatting

Learn: Paragraphs use <p>. Use <strong> for bold and <em> for italics.

See:

<p>I enjoy <strong>coding</strong> and <em>learning new things</em>.</p>

Do: Write two paragraphs about a hobby or interest using bold and italics naturally.

Check: Make sure formatting highlights important words correctly.

Day 4: Lists

Learn: Lists organize content. <ul> for bullets, <ol> for numbers.

See:

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<ol>
  <li>Wake up</li>
  <li>Study</li>
</ol>

Do: Make one unordered list of hobbies and one ordered list of daily routine.

Check: Verify bullets and numbers show correctly and try nesting a sub-list.

Day 5: Links

Learn: Links navigate to other pages. Use href for destinations.

See:

<a href="https://example.com">Visit Example</a>

Do: Add two external links and one link to a local file.

Check: Hover and click links to make sure they work.

Day 6: Images

Learn: Images use <img>. Alt text is important for accessibility.

See:

<img src="image.jpg" alt="Description" width="200">

Do: Add an image and adjust width.

Check: Ensure the image loads and alt text is meaningful.

Day 7: Review & Mini Profile

Learn: Combine all skills into a profile page.

Do: Include:

  • H1 heading
  • Two paragraphs
  • One unordered list
  • One ordered list
  • One image
  • One link

Check: Verify display, indentation, and readability in a browser.

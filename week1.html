Week 1: HTML Basics

Welcome to Week 1! This week, you will learn the foundations of HTML using an interactive method: Learn the concept, See an example, Do a small exercise, and Check your results in the browser. Each day is designed to take about 30 minutes.

Day 1: HTML Overview & Page Structure

Learn: HTML is the foundation of web pages. Every document starts with <!DOCTYPE html> and is wrapped in <html>. The <head> contains metadata like <title> and <meta charset="UTF-8">, while the <body> contains all visible content. See: Example of a simple page:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first paragraph.</p>
</body>
</html>
Do: Create a new HTML file with your name as the <h1> heading and a paragraph about yourself. Check: Open it in a browser and verify that the heading and paragraph appear correctly, then try changing the text to see updates instantly.

Day 2: Headings

Learn: Headings organize content using <h1>–<h6>, with <h1> as the main title. Logical heading levels improve readability and accessibility. See:

<h1>My Website</h1>
<h2>About Me</h2>
<h3>Hobbies</h3>
Do: Create a mini-bio page using H1 for your name, H2 for “About Me,” and H3 for “Hobbies,” adding a few sentences under each. Check: Verify that the headings appear in decreasing size and are logically structured, and use only one <h1> per page.

Day 3: Paragraphs & Text Formatting

Learn: Paragraphs use <p> and text can be emphasized with <strong> for bold and <em> for italics. See:

<p>I love <strong>coding</strong> and <em>reading</em> books.</p>
Do: Write two paragraphs about yourself or your hobby, including bold and italic text. Check: Open in a browser to verify formatting, making sure important words are emphasized correctly and the page is readable.

Day 4: Lists

Learn: Lists organize content: <ul> for bullets, <ol> for numbered items, and <li> for each item. Lists can also be nested. See:

<h2>Hobbies</h2>
<ul>
  <li>Coding</li>
  <li>Gaming</li>
</ul>
<h2>Daily Routine</h2>
<ol>
  <li>Wake up</li>
  <li>Code</li>
</ol>
Do: Create your own unordered list of hobbies and an ordered list of daily routine. Check: Open in a browser to see bullets and numbers, and experiment with nesting a sub-list inside a list item.

Day 5: Links

Learn: Links are created with <a href="URL">Link Text</a> and allow navigation to other pages or websites. See:

<p>Visit <a href="https://www.wikipedia.org">Wikipedia</a> for information.</p>
Do: Add at least two links to your favorite websites with descriptive text, and try linking to a local HTML page like day1.html. Check: Hover over links to confirm they are clickable and open the correct destination.

Day 6: Images

Learn: Images are added with <img src="file.jpg" alt="description"> and can include width and height attributes. Alt text is important for accessibility. See:

<img src="profile.jpg" alt="My Profile Picture" width="200">
Do: Add a profile picture or placeholder image to your page, set a width for proper display, and add meaningful alt text. Check: Open in a browser to see the image and experiment with different placements relative to text.

Day 7: Review & Mini Profile

Learn: Combine all the skills from Days 1–6 to create a complete HTML profile page, including H1 heading, two paragraphs, an unordered list, an ordered list, one link, and one image. Do: Build the page, paying attention to indentation, semantic structure, and readability. Check: Open in a browser to verify that all elements display correctly, are logically structured, and your code is clean, making adjustments if necessary. This final exercise consolidates all your HTML basics for Week 1.

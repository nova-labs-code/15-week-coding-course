Week 1: HTML Basics

Welcome to Week 1! This week focuses on the foundations of HTML. HTML is used to structure content on the web, and everything you see on a website starts with it. Each day follows the same pattern: Learn the idea, see a real example, try it yourself, and check the result in your browser. Each lesson is designed to take about 30 minutes.

Day 1: HTML Overview & Page Structure

Learn:
HTML stands for HyperText Markup Language. It uses tags to describe content like headings, paragraphs, images, and links. Every HTML file has a required structure that browsers expect.

See:

<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>My First Page</title>
</head>
<body>
  <h1>Hello World</h1>
  <p>This is my first paragraph.</p>
</body>
</html>

Do:
Create a new HTML file. Add your name inside an <h1> tag and write one paragraph about yourself.

Check:
Open the file in a browser. Change the text and refresh the page to see the update immediately.

Tip: Browsers ignore extra spaces. Clean indentation helps you read your code, not the browser.
Day 2: Headings

Learn:
Headings range from <h1> to <h6>. They create structure and hierarchy on a page. Only one <h1> should be used per page.

See:

<h1>My Website</h1>
<h2>About Me</h2>
<h3>Hobbies</h3>

Do:
Create a page with your name as an H1, an “About Me” section using H2, and a “Hobbies” subsection using H3.

Check:
Confirm headings appear in decreasing size and make sense when read top to bottom.

Day 3: Paragraphs & Text Formatting

Learn:
Paragraphs are created with <p>. Use <strong> for importance and <em> for emphasis.

See:

<p>I enjoy <strong>coding</strong> and <em>learning new things</em>.</p>

Do:
Write two paragraphs about a hobby or interest. Use bold and italic text naturally.

Check:
Make sure formatting highlights important words without overusing it.

Day 4: Lists

Learn:
Lists organize information clearly. Use <ul> for unordered lists and <ol> for ordered lists.

See:

<ul>
  <li>HTML</li>
  <li>CSS</li>
</ul>

<ol>
  <li>Wake up</li>
  <li>Study</li>
</ol>

Do:
Create one unordered list of hobbies and one ordered list of your daily routine.

Check:
Verify bullets and numbers display correctly and try nesting a list inside another.

Day 5: Links

Learn:
Links connect pages and websites. The href attribute defines the destination.

See:

<a href="https://example.com">Visit Example</a>

Do:
Add two external links and one link to another local HTML file.

Check:
Hover over links and click them to ensure they go to the correct place.

Day 6: Images

Learn:
Images use the <img> tag. The alt attribute is required for accessibility.

See:

<img src="image.jpg" alt="Description" width="200">

Do:
Add an image to your page and resize it using the width attribute.

Check:
Confirm the image loads and the alt text makes sense.

Day 7: Review & Mini Profile

Learn:
Today you combine everything you learned this week into one clean HTML page.

Do:
Create a profile page that includes:

  • One H1
  • At least two paragraphs
  • An unordered list
  • An ordered list
  • One image
  • One link

Check:
Open the page in a browser and review structure, spacing, and readability. Clean up indentation and make sure everything makes sense visually.

Week 12: JS Arrays & Dynamic Pages

This week introduces **arrays and dynamic page updates**, allowing multiple values to be stored, iterated, and displayed on your page.

Day 1: Creating Arrays

Learn: Arrays store multiple values. See:

let fruits = ["apple", "banana", "orange"];
console.log(fruits);
Do: Create an array with your top 5 favorite foods. Check: Console logs all items correctly.

Day 2: Accessing & Modifying Arrays

Learn: Access items by index, modify values. See:

console.log(fruits[1]); // banana
fruits[0] = "mango";
Do: Change the first item in your array and log it. Check: Browser console shows updated value.

Day 3: Array Methods (push, pop, shift, unshift)

Learn: Add/remove elements easily. See:

fruits.push("grape"); // add end
fruits.pop(); // remove end
fruits.unshift("kiwi"); // add start
fruits.shift(); // remove start
Do: Manipulate your array and log changes each time. Check: Console reflects modifications.

Day 4: Looping Through Arrays

Learn: Use loops to process array items. See:

for (let i = 0; i < fruits.length; i++) {
  console.log(fruits[i]);
}
Do: Log all items of your array with a message like "I like ___". Check: Browser console shows correct output.

Day 5: Display Array Items on Page

Learn: Use DOM to display array content dynamically. See:

let list = document.getElementById("myList");
fruits.forEach(function(fruit) {
  list.innerHTML += "<li>" + fruit + "</li>";
});
Do: Create <ul id="myList"></ul> and dynamically add array items. Check: Browser shows list of items.

Day 6: Interactive Array Updates

Learn: Use input fields to add array items. See:

<input id="fruitInput">
<button onclick="addFruit()">Add</button>

<script>
let fruits = [];
function addFruit() {
  let val = document.getElementById("fruitInput").value;
  fruits.push(val);
  console.log(fruits);
}
</script>
Do: Create input and button to add items to your array dynamically. Check: Browser console shows updated array.

Day 7: Review & Mini Dynamic Page

Learn: Combine arrays, loops, and DOM updates. Do: Build a simple dynamic shopping list: enter items, click "Add", and see them appear on the page. Check: Browser updates correctly. This completes Week 12.

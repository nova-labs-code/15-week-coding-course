Week 6: CSS Advanced – Layout & Transitions

This week we dive into **advanced CSS techniques**: transitions, transforms, layering, opacity, shadows, and interactive hover effects. Each day now contains multiple examples, exercises, and tips to practice and understand smoothly.

Day 1: CSS Transitions

Learn: CSS transitions allow smooth animation between property changes. You can transition colors, sizes, spacing, shadows, and more. Example: transition: all 0.3s ease; will animate any change over 0.3 seconds.

See:

div.box {
  background: #ddd; 
  padding: 20px; 
  transition: all 0.3s ease;
}
div.box:hover {
  background: #aaa;
  transform: scale(1.05);
}

Do: Apply a hover transition to a section or card on your page, try changing background color, padding, or scale.

Check: Hover in the browser to see smooth animation.

Tip: Use transition for subtle changes; too many can make a page feel slow.
Day 2: Transform & Scale

Learn: CSS transform allows scaling, rotating, moving, or skewing elements. Example: transform: scale(1.1); makes an element 10% larger on hover.

See:

div.card {
  transition: transform 0.3s ease;
}
div.card:hover {
  transform: scale(1.1) rotate(5deg);
}

Do: Apply scale and rotation to a card or image on hover. Experiment with values like 1.2, -10deg rotation, or translateX/Y to move elements slightly.

Check: Hover in browser and see interactive growth or movement.

Tip: Small transforms give a subtle, professional effect; large transforms may feel jarring.
Day 3: Z-index & Layering

Learn: z-index controls stacking order of elements. Only works on positioned elements (relative, absolute, fixed, sticky). Higher numbers appear on top.

See:

div.background { position: relative; z-index: 1; }
div.foreground { position: relative; z-index: 2; }

Do: Overlap two divs and assign different z-index values to see which appears on top. Experiment with negative z-index for background layering.

Check: Browser displays correct layering.

Tip: Use z-index carefully; too many layers can confuse layout structure.
Day 4: Opacity & Visibility

Learn: Control element transparency with opacity (0 fully transparent, 1 fully visible) and toggle visibility with visibility. Example: opacity: 0.5; makes element semi-transparent.

See:

div.overlay {
  opacity: 0.7;
  transition: opacity 0.3s ease;
}
div.overlay:hover {
  opacity: 1;
}

Do: Add opacity to an image or card and adjust on hover. Combine with transitions for smooth effect.

Check: Browser shows fading effect when hovering.

Tip: Opacity can enhance layering and focus without removing elements.
Day 5: Hover Transitions Combined

Learn: Combine transformations, color changes, opacity, and shadow in one smooth hover effect.

See:

div.card {
  transition: all 0.4s ease;
  background: #eef;
  box-shadow: 2px 2px 5px rgba(0,0,0,0.2);
}
div.card:hover {
  transform: scale(1.05) rotate(-2deg);
  opacity: 0.85;
  background-color: #cce7ff;
  box-shadow: 4px 4px 10px rgba(0,0,0,0.3);
}

Do: Apply combined hover effects to a section, button, or card. Experiment with scaling, rotation, opacity, and shadow simultaneously.

Check: Observe smooth visual change in browser.

Tip: Combining subtle effects creates a professional and interactive feel.
Day 6: Border & Shadow Animations

Learn: Animate borders and shadows using transitions for dynamic UI feedback. Example: hover highlights card with shadow and border color change.

See:

div.card {
  border: 2px solid #aaa;
  border-radius: 10px;
  box-shadow: 2px 2px 5px rgba(0,0,0,0.2);
  transition: all 0.3s ease;
}
div.card:hover {
  border-color: #1e90ff;
  box-shadow: 4px 4px 15px rgba(0,0,0,0.4);
}

Do: Apply border and shadow transitions to a card or image section. Experiment with color, shadow size, and border width.

Check: Browser shows animated border and shadow on hover.

Tip: Use these animations for call-to-action elements or highlighting important content.
Day 7: Review & Interactive Page

Learn: Merge all Week 6 techniques: transitions, transforms, z-index, opacity, shadows, and hover effects to create an interactive profile page or card layout.

Do:

  • Enhance at least three sections/cards with hover effects and transitions.
  • Use layering, opacity, and shadows to guide visual hierarchy.
  • Experiment with scale, rotation, and color transitions for interactivity.
  • Ensure consistent design, smooth animations, and readability.

Check: Open in browser to verify all effects work seamlessly and the page feels dynamic and professional.

Tip: Subtle, consistent effects improve user experience without overwhelming the design.

Week 10: JS Functions, Events & DOM

This week focuses on **functions, events, and basic DOM manipulation**, giving your page interactivity beyond console logging.

Day 1: Functions

Learn: Functions are reusable blocks of code. See:

function greet(name) {
  console.log("Hello " + name);
}
greet("Elisha");
Do: Create a function to add two numbers and log the result. Check: Console shows correct sum.

Day 2: Parameters & Return Values

Learn: Functions can take inputs and return outputs. See:

function square(num) {
  return num * num;
}
console.log(square(5)); // 25
Do: Create a function that multiplies two numbers and returns the result. Check: Console prints correct value.

Day 4: DOM Selection

Learn: JS can select HTML elements using getElementById, getElementsByClassName, querySelector. See:

let header = document.getElementById("header");
header.style.color = "red";
Do: Give a heading an ID and change its color using JS. Check: Open browser to verify color changes.

Day 5: DOM Content Change

Learn: JS can change content with innerText or innerHTML. See:

let msg = document.getElementById("msg");
msg.innerText = "Welcome!";
Do: Create a paragraph with an ID and change its text on button click. Check: Browser updates text correctly.

Day 6: Combining Events & DOM

Learn: Combine click events with DOM updates. See:

<button onclick="changeText()">Click</button>
<p id="demo">Old Text</p>

<script>
function changeText() {
  document.getElementById("demo").innerText = "New Text!";
}
</script>
Do: Make a button change the text of a paragraph. Check: Verify behavior in browser.

Day 7: Review & Mini Interactive Page

Learn: Combine functions, events, and DOM manipulation for interactivity. Do: Create a small interactive profile card with a button that updates a message or favorite hobby when clicked. Check: Open browser and test button behavior. This completes Week 10.
